## 2. Que se esta decidiendo realmente

Estas lineas no premian que una startup tenga software. Premian que una pyme ejecute un salto de digitalizacion o transformacion concreto y elegible.

Por eso TradeBase tiene un riesgo especial: ser demasiado nativa digital. Si el expediente no aísla un salto identificable, el evaluador puede leer que no hay proyecto de digitalizacion, sino operativa normal de una software company.

---

## 3. Requisitos que importan

### 3.1 Convocatoria viva correcta

No mezclar familias. Cada convocatoria puede cambiar:

- intensidad;
- gasto minimo;
- categorias elegibles;
- y fecha limite.

### 3.2 Proyecto no concluido materialmente

Esto es clave. No sirve pedir una ayuda para algo ya implantado o completamente ejecutado.

### 3.3 Salto de digitalizacion identificable

Debe existir un antes y un despues claros.

---

## 4. Como podria encajar en TradeBase

Solo con bloques como:

- automatizacion avanzada del canal dealer;
- mejora estructural de procesos internos;
- implantacion de analitica y automatizacion no existente previamente;
- digitalizacion operativa asociada a crecimiento de actividad real.

---

## 5. Que no meteria

- nucleo basico de desarrollo del producto;
- tareas ordinarias de una startup software;
- gastos cerrados o ya concluidos;
- mezcla difusa de herramientas sin salto concreto.

---

## 6. Documentacion minima

- mapa del proceso actual;
- paquete de transformacion definido;
- presupuesto separado;
- cronograma con parte pendiente real;
- soporte societario y territorial.

---

## 7. GO / NO-GO / NOT YET

### GO

Si existe convocatoria viva y el paquete de transformacion es claramente distinto del trabajo ordinario de producto.

### NO-GO

Si la linea se intenta usar para subvencionar el core normal de una software company.

### NOT YET

Si falta aislar el salto de digitalizacion concreto.

---

## 8. Conclusion operativa

A4 puede ser util, pero obliga a hacer un trabajo fino: convertir una compania digital en un expediente de transformacion identificable. Si no se hace esa separacion, la linea pierde fuerza.
