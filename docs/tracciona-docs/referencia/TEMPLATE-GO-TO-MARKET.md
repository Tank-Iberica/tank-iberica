## 9. Timeline de lanzamiento

| Semana | Hito |
|--------|------|
| -4 | Vertical creada técnicamente, seed data, branding |
| -3 | Contenido SEO publicado, dealers founding contactados |
| -2 | Beta privada con 5-10 dealers, QA completo |
| -1 | Pre-launch checklist completada, monitoring activo |
| 0 | **Go-live** — anuncio en redes, email a base |
| +1 | Seguimiento dealers, primeros leads, ajustes UX |
| +2 | Análisis métricas, iteración rápida |
| +4 | Review KPIs mes 1, planificar mes 2 |

## 10. Riesgos y mitigación

| Riesgo | Probabilidad | Impacto | Mitigación |
|--------|-------------|---------|------------|
| Pocos dealers al inicio | Alta | Alto | Programa founding + cold outreach |
| SEO tarda en posicionar | Alta | Medio | SEM temporal + contenido agresivo |
| Competidor reacciona | Media | Medio | Diferenciación en servicios |
| Problemas técnicos | Baja | Alto | Pre-launch checklist + monitoring |
