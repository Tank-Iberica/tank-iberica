Tomo A1 - SME Fund EUIPO

Rol: monografia operativa para decidir cuando tiene sentido usar SME Fund y como evitar pedirlo sin un calendario real de propiedad industrial o marca.
Fecha de corte: 11/03/2026.
Fuente principal: https://euipo.europa.eu/sme-fund-2026/

1. Ficha de trabajo

Campo Detalle
Organismo EUIPO
Naturaleza Bono / reembolso sobre gastos de PI
Cuello de botella TradeBase No sirve si el registro de marca, diseno o patente no esta calendarizado
Decision actual GO cuando exista accion real de PI; NO-GO mientras sea idea vaga

2. Que se esta decidiendo realmente

No es una linea estrategica por si sola. Es una herramienta tactica para abaratar un registro que ya ibais a hacer. Por eso no tiene sentido tratarla como si fuera una ayuda de expansion o de producto.

3. Que tiene sentido para TradeBase

  • marca TradeBase;
  • marca Tracciona;
  • eventual marca de CampoIndustrial si se activa;
  • diseno o activo adicional si la estrategia de PI madura.

4. Que no tiene sentido

  • pedir el bono sin decision real de registro;
  • usarlo como excusa para hablar de PI que todavia no tiene calendario;
  • retrasar un registro urgente por esperar una ventana dudosa.

5. Documentacion minima

  • titularidad societaria clara;
  • clases a registrar definidas;
  • calendario de presentacion;
  • presupuesto de tasas y servicios;
  • control de no solape con otros gastos.

6. GO / NO-GO / NOT YET

GO

Si existe registro decidido y calendario de solicitud.

NO-GO

Si la PI sigue sin fecha ni prioridad real.

NOT YET

Si la marca interesa, pero aun no esta cerrada la estrategia de denominaciones y clases.

7. Conclusion operativa

A1 es util cuando ya habeis decidido registrar. Antes de eso no es una ayuda; es solo ruido.
